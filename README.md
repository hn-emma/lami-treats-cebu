# 🌴 Lami Treats Cebu - Custom Shopify Theme

A custom Shopify Online Store 2.0 theme built for **Lami Treats Cebu**, a storefront specializing in local Cebuano *pasalubong*. This project is developed as part of a technical training initiative, focusing on modern e-commerce development, seamless user experiences, and clean code architecture.

## 🛠️ Tech Stack

- **Platform:** Shopify (Online Store 2.0)
- **Templating:** Liquid, JSON Templates
- **Styling:** Tailwind CSS
- **Interactivity:** Vanilla JavaScript (ES6 Classes, Web Components / Custom Elements, Fetch API)
